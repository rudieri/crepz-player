Crepz Player Tutorial

O Crepz Player é um programa pra execução de arquivos de áudio, foi desenvolvido como trabalho final de Linguagem de Programação II na Unijui.

A Tela Principal

A tela principal possui os botões para gerenciamento das musica em execução e botões para chamada das telas secundárias.


Teclas de atalho:
  • Tecla B: Chama a Biblioteca de Midia
  • Tecla P: Chama a Playlist
  • F4 : Minimiza para A bandeja (Quando estiver na bandeja, você terá uma mini tela para mexer)

Biblioteca

Na Biblioteca o Usuário pode gerenciar suas musicas, através do autor, álbum. Para importação de novos arquivos, deve-se selecionar uma pasta com os componentes no rodapé da janela.


Teclas de atalho:
  • Tecla F5: Filtra o Resultado
  • Tecla F4: Limpa os Filtros
  • Tecla P : Adiciona Musica selecionada Para a Playlist
  • Tela T: Adiciona Todas Para a Playlist

Playlist

Na Playlist pode-se gerenciar as musica que serão executadas. Ainda se tem o recurso de exportar para um arquivo .mpu pra execução da playlist e outros players.


Teclas de atalho:
  • Ctrl + S : Salva a Playlist Atual
  • Tecla F4: Limpar
  • Tecla F5: Consultar Playlist
  • Tecla F6: Delete
